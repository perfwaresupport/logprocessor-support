# Log Processor — Support

Public issue tracker for [Log Processor](https://perfware.cloud) on AWS Marketplace.

## Reporting Issues

- **Bug reports**: [Open an issue](https://github.com/perfwaresupport/logprocessor-support/issues/new?template=bug_report.md)
- **Feature requests**: [Open an issue](https://github.com/perfwaresupport/logprocessor-support/issues/new?template=feature_request.md)
- **Questions**: [Start a discussion](https://github.com/perfwaresupport/logprocessor-support/discussions)

## Before Reporting

1. Check [existing issues](https://github.com/perfwaresupport/logprocessor-support/issues) for duplicates
2. Include your deployment tier
3. Include relevant CloudWatch log errors if applicable
4. Do not include AWS credentials, account IDs, or sensitive data

## Support Channels

| Channel | Use For | Response |
|---|---|---|
| [GitHub Issues](https://github.com/perfwaresupport/logprocessor-support/issues) | Bugs, feature requests | 2 business days |
| support@perfware.cloud | Configuration help, account issues | 2 business days |
| consulting@perfware.cloud | Custom integration, managed service | By arrangement |
